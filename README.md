# FontMaker
点阵字体在嵌入式GUI开发中，使用非常广泛。

主要有两种使用场景：

- 生成某种编码的全部点阵，以BIN的形式放在FLASH中加载。
- 生成部份文字的点阵，以C文件的方式，写在代码里。


本工具的特点：

- 完全免费，不用到处找破解工具;
- 通用性好，支持任意编码，任意语言，可以自定义编码;
- 可以生成BIN和C文件;
- 可以使用WINDOWS全部字体;
- 使用方便，界面整洁;
